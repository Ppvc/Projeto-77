Projeto no console

O Visual Studio Code é um editor de código-fonte desenvolvido pela Microsoft para Windows, Linux e macOS. Ele inclui suporte para depuração, controle de versionamento Git incorporado, realce de sintaxe, complementação inteligente de código, snippets e refatoração de código, e pode ser consigurado de várias maneiras e de acordo com seu estilo.

Ela é a janela para a interação direta entre um usuário e um sistema de computador, proporcionando uma maneira textual de comunicar com a máquina, ela poder ser visualizada se você clicar com o botão direito do mouse, clicar em "Inspecionar" e ir para a aba "Console", para utilizá-la basta escrever o código "console.log".
